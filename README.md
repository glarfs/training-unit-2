Training Unit 2
===============

This is the repository of the Training Unit 2 subproject. It provides an starting point for an Appverse Web HTML5 project
Appverse Web is a multi-frontend technology capable web application framework incorporating the best-of-breed open source stack to meet the challenges of secure and robust server-side service development, integration and delivery.

## Step by Step guideline

There are a few tags on the following repository:

* **init**: Initial step to start up with bower components and initial index.html file and grunt tasks
* **step1**: create an initial home page with the required dependencies
* **step2**: create tasks view and controller and a back button
* **step3**: create a REST call to json file with 3 elements and load it to Tasks view with the following fields title(String) and done(true or false)
* **step4**: show a list with list contents and show also a checkbox with 'done' state (true checked, false unchecked)
* **step5**: add field with filter action
* **step6**: add add and remove functionality
* **step7**: i18n on Home and Tasks title
* **final**: All completed steps

### Quick Start

#### Before you start, tools you will need

* install npm
* bower and grunt (run the following commands):

```script
npm install -g bower
npm install -g grunt
```

## Running

* configure project:

```script
npm install
bower install
```
* run project

`grunt server`

## Testing

Exexecute the following command to launch tests

`grunt test`
